Uso de la API del Covid-19 con Pandas¶

En esta prctica vamos a utilizar "Pandas", la librera de Phyton, para trabajar sobre los datos de la Covid-19 y analizarlos.

La URL: https://api.covid19api.com/countries

Configuración de Pandas¶

Para importar utilizaremos la convencin "pd" como abrebiatura para llamar a la librera.

In [2]:
import pandas as pd

Crear variable¶

Las variables se asignan con el smbolo "=" y los escribimos entrecomillados por ser una cadena de caracteres.

In [3]:
url = "https://api.covid19api.com/countries"

Crear dataframe¶

La abreviatura que se emplea para los dataframe es "df". Existe una funcin "read_json()" que lo que hace es leer el formato "json". Dentro del parntesis indicaremos el valor de lo que queremos que lea, en este caso la url.

In [4]:
df = pd.read_json(url)
